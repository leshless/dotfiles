# Dotfiles

Dotfile configs for various CLIs and services on my personal car.

All configs are linked to original locations via [GNU Stow](https://www.gnu.org/software/stow).

To create symlinks run:
```bash
cd dotfiles && stow .
```

This meant to be used with my [second repository](https://github.com/leshless/scripts) with scripts and shorthands.
